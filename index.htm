API Basejump: Image Search Abstraction Layer


Usage tips


-As a parameter string of URL passed. String entered after /api/imgsearch/"your search string" will be a search string.

-If "offset=2" parameter is added afer search string like this /api/imgsearch/"your search string"/offset=2 then response object will be paginated.

-If "latest" parameter is added afer imgsearch like this /api/imgsearch/latest then most recent search results will be shown.

Example of how URL string should be passed as a parameter:


https://imgsearchabslayer.herokuapp.com/api/imgsearch/gold fish

To paginate it:


https://imgsearchabslayer.herokuapp.com/api/imgsearch/gold fish/offset=2

Getting latest search result:


https://imgsearchabslayer.herokuapp.com/api/imgsearch/latest
